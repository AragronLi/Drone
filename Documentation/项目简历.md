### 项目概况

自主研发的四旋翼无人机飞行控制器，基于沁恒 **CH32H417 双核 RISC-V MCU**（青稞 V3F 120MHz + V5F 480MHz），实现从底层传感器采集到上层 MAVLink 地面站通信的**全栈飞控系统**。

| 项目周期 | 2025/12 — 2026/05 |
|---|---|
| 角色 | 个人全栈开发（硬件选型、嵌入式固件、通信协议、地面站） |
| 代码量 | ~3500 行 C 固件 + ~800 行 Python 地面站 |
| 硬件平台 | CH32H417QEU6 (QFN128) + CH32V203 ESC ×4 |
| 开源协议 | MAVLink v2 |

---

### 核心技术亮点

**1. 异构双核实时调度**
V3F 小核（裸机）负责全传感器采集与 PWM 输出，V5F 大核（FreeRTOS）负责姿态解算与控制算法，两核通过 IPC + HSEM 硬件信号量实现零拷贝数据交换，心跳超时 500ms 自动触发安全锁定。

**2. 多传感器融合**
集成 ATK 十轴 IMU（200Hz）、下视激光/前向双路 TOF（100Hz）、PMW3901 光流传感器（100Hz SPI）、GPS + 罗盘旧方案，实现 TOF-气压互补定高与光流-IMU 倾角补偿室内定位。

**3. MAVLink v2 全协议通信**
基于 pymavlink 工具链生成 C 头文件，支持 HEARTBEAT、ATTITUDE、RAW_IMU、ALTITUDE、BATTERY_STATUS 等 8 类遥测帧 80ms 周期轮转发送，兼容 QGroundControl 等标准地面站。

**4. 自研 Python 地面站**
基于 tkinter + pymavlink + matplotlib 的 PC 端地面控制站，支持三维姿态仪表盘、6 通道实时波形示波器、ARM/DISARM 远程控制与飞行模式切换。

**5. 硬件系统设计**
完整的飞控核心板设计，集成 SY8088AAC 开关降压 + TPS54202DDCR 3.3V 电源，混合使用 GH1.25、XH2.54、HDR-M_2.54、SH1.0 等连接器，支持 GPS/光流/TOF/数传/空闲/预留/OLED/SD 卡等模块化扩展。

### 光流定位算法

**倾角补偿 + 高度换算 + 低通滤波**

```c
// 1. 倾角补偿: 消除机体倾斜带来的像素偏移
pixComp[X] = 480.0 * tan(pitch);   pixComp[Y] = 480.0 * tan(roll);

// 2. 像素 → 位移 (高度来自 TFmini TOF)
deltaPos = 0.2132 * height_m * (pixValid - pixValidLast);

// 3. 瞬时速度 + 一阶低通
velLpf += (deltaPos / dt - velLpf) * 0.15;
```

- 有效高度 5cm ~ 4m，自动故障检测（maxRaw/minRaw 连续为 0 > 1s）
- 支持 X 型四旋翼方向适配（pitch → X, roll → Y）

### 安全设计

- **双核互锁看门狗**: V3F 检测 V5F 心跳超时 500ms → 强制停转
- **失控保护**: 遥控器断线 >1s 且 GPS 有效 → 自动 RTL 返航；GPS 无效 → 原地降落
- **前向避障**: 前视激光 渐进减速，< 0.8m 强制悬停（所有模式生效）
- **电池保护**: 低电 LED 闪烁 + MAVLink 告警 → 强制 RTL 降落
- **光流故障检测**: 连续 1s 无效数据 → 标记传感器异常，切换纯 IMU 模式
- **SWD 设计**: PB8/PB9 专用于烧录调试；旧 OLED 已停用；罗盘走 I2C3 PA13/PA14，GPS 走 USART4 PC6/PC7，不与 SWD 冲突

### 开发调试经验

1. **SWD 锁死问题**: 排查发现 `GPIO_Remap_SWJ_Disable` 在 CH32H417 上会禁用全部 SWJ（含 SWD），导致芯片不可编程。解决：移除该调用，SPI3 的 AF6 配置本身即可覆盖 JTAG 默认功能，无需禁用。
2. **双核工程配置**: MounRiver Studio 中 V3F/V5F 各自独立工程，共享 Common 目录通过 `.project` 链接资源实现。
3. **MAVLink 工具链**: `pymavlink` 从 `common.xml` + `standard.xml` 生成 C 头文件，关键宏 `MAVLINK_USE_CONVENIENCE_FUNCTIONS` 启用便捷打包函数。
4. **光流标定**: RESOLUTION (0.213) 和 COMP_COEFF (480) 需在 1m 高度实测标定，不同模块与镜头略有差异。
