# CH32H417QEU6 封装说明

> **串口分配最终确认（2026-05-31）**
> 本项目硬件分配最终以此为准：USART1=Debug(PB6/PB7)、USART2=接收机(PA3)、USART3=陀螺仪(PB10/PB11)、USART4=GPS(PC6/PC7)、USART5=PF5/PE0空闲、USART6=下视激光(PC10/PC11)、USART7=前视激光(PC12/PD2)、USART8=数传(PE8/PE7)、I2C3=罗盘(PA14=SCL/PA13=SDA)。


## QFN128封装特性

### 基本参数
- **封装类型**: QFN128 (Quad Flat No-lead)
- **尺寸**: 14mm × 14mm
- **引脚间距**: 0.4mm
- **引脚数量**: 128引脚
- **底部散热焊盘**: 有 (增强散热)

### 与LQFP-64的对比

| 特性 | LQFP-64 | QFN128 |
|------|---------|--------|
| 引脚数 | 64 | 128 |
| 封装尺寸 | 10×10mm | 14×14mm |
| 引脚间距 | 0.5mm | 0.4mm |
| 引脚类型 | 鸥翼型 | 无引脚 |
| 散热性能 | 一般 | 优秀 |
| GPIO数量 | 较少 | 更多 |
| 外设接口 | 基本 | 完整 |
| 焊接难度 | 容易 | 较难 |

### QFN128的优势

1. **更多GPIO**: 128引脚提供更多可用GPIO
2. **完整外设**: 支持更多USART、SPI、I2C、TIM等外设
3. **更好散热**: 底部散热焊盘，适合高性能应用
4. **更小体积**: 相对引脚数，封装更紧凑
5. **更多扩展**: 预留更多接口用于功能扩展

### QFN128引脚分布

```
        ┌─────────────────────────┐
        │  32引脚 (顶部)           │
    ┌───┤                         ├───┐
    │32 │                         │ 32│
    │引 │    CH32H417QEU6         │ 引│
    │脚 │      QFN128             │ 脚│
    │   │    14mm × 14mm          │   │
    │左 │                         │ 右│
    │侧 │    [散热焊盘]           │ 侧│
    └───┤                         ├───┘
        │  32引脚 (底部)           │
        └─────────────────────────┘
```

### 焊接注意事项

1. **回流焊接**: 推荐使用回流焊，手工焊接困难
2. **焊盘设计**: 需要精确的PCB焊盘设计
3. **散热焊盘**: 底部散热焊盘必须焊接并连接到GND
4. **检查方法**: 使用X-ray检查焊接质量
5. **返修困难**: QFN封装返修需要专业设备

### PCB设计建议

1. **焊盘尺寸**: 严格按照数据手册设计
2. **散热过孔**: 在散热焊盘下方打多个过孔到底层GND
3. **阻焊设计**: 引脚焊盘开窗，散热焊盘可选择性开窗
4. **丝印标识**: 标注第1脚位置和芯片方向
5. **测试点**: 关键引脚引出测试点

### CH32H417QEU6引脚资源

相比LQFP-64版本，QFN128版本额外提供：
- 更多GPIO引脚
- 完整的USART接口 (USART1-8)
- 完整的SPI接口 (SPI1-3)
- 完整的I2C接口 (I2C1-3)
- 更多定时器通道
- 更多ADC通道
- 完整的CAN接口 (CAN1-3)
- USB HS接口

### 本项目使用的外设

| 外设 | 数量 | 使用情况 |
|------|------|---------|
| USART | 8个 | 已使用8个 (满载) |
| SPI | 3个 | 已使用2个 (SPI2, SPI3) |
| I2C | 3个 | 已使用1个 (I2C3) |
| TIM | 多个 | 已使用TIM1 |
| ADC | 多个 | 已使用ADC1_CH1 |
| CAN | 3个 | 预留CAN1 |
| GPIO | 大量 | 充足 |

### 总结

CH32H417QEU6 (QFN128) 为本飞控项目提供了充足的引脚资源，支持：
- 8路USART (IMU, TOF×2, GPS, 数传, 空闲/预留, IBUS, Debug)
- 2路SPI (光流, SD卡)
- 1路I2C (GPS 串口共享)
- 4路PWM (ESC)
- 多路GPIO (蜂鸣器, LED, 使能控制等)
- 预留扩展接口 (SPI1, CAN1, GPIO)

QFN128封装是本项目的理想选择，提供了足够的接口资源和良好的散热性能。
