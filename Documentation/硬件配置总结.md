## 一、核心硬件配置

### 1.1 MCU
- **型号**: CH32H417QEU6 (QFN128封装)
- **封装**: QFN128 (14mm × 14mm, 0.4mm间距, 128引脚)
- **双核架构**: 
  - V3F (小核): 120MHz, 裸机
  - V5F (大核): 480MHz, FreeRTOS
- **晶振**: 25MHz外部晶振 (X2)
- **Flash地址**: V3F @ 0x00000000, V5F @ 0x00010000
- **优势**: QFN128提供更多GPIO和外设资源

### 1.2 电源系统
```
VBAT (4S LiPo, 14.8~16.8V)
  │
  ├─→ SY8088AAC (U12) → 5V/2A → 外设供电
  │
  ├─→ SY8088AAC (U14) → 5V
  │    └─→ TPS54202DDCR (U15) → 3.3V (VDD33) → 数字电源
  │
  └─→ MPZ1608S601ATA00 (L6) → VDD33A → 模拟电源 (ADC/DAC)
```

## 三、关键硬件特性

### 3.1 光流传感器 (PMW3901)
- **接口**: SPI3 (PB3/4/5) + CS(PC5)
- **特性**: 33Ω串联电阻 (R170/171/172) 用于信号完整性
- **连接器**: GH1.25-6P
- **速率**: 2MHz SPI时钟
- **有效高度**: 5cm ~ 4m

### 3.2 电池电压检测
- **引脚**: PA1 (ADC1_IN1)
- **电路**: VBAT → R179(10kΩ) → PA1 → R180(2kΩ) → GND
- **分压比**: 1:6
- **滤波**: C105 (100nF)
- **计算公式**: 
```c
float vbat = (adc_value * 3.3 / 4096.0) * 6.0;
```
- **测量范围**: 0 ~ 19.8V (4S满电16.8V → 2.8V ADC输入)

### 3.3 PWM输出
- **定时器**: TIM1 CH1-4
- **引脚**: PA8/9/10/11
- **频率**: 50Hz (20ms周期)
- **脉宽**: 1000μs (停转) ~ 2000μs (全速)
- **保护**: 每路100Ω下拉电阻 (R175-178)，防止上电误触发
- **连接器**: SH1.0-6P

### 3.4 GPS 串口分配
- **USART4 GPS (PC6/PC7)**: 共享总线
  - GPS NMEA 数据
  - 115200, 8N1
  - PA13/PA14 作为 I2C3 罗盘使用，旧 OLED 停用

### 3.5 SWD烧录接口
- **引脚**: PB8 (SWCLK), PB9 (SWDIO)
- **独立接口**: 不与I2C冲突
- **连接器**: HDR-M_2.54-4P
- **注意**: I2C1 (PB8/9) 未使用；PA13/PA14 用于 I2C3 罗盘；GPS 使用 USART4 PC6/PC7；旧 OLED 停用

### 5.2 I2C配置
```c
// I2C3 罗盘总线 - QMC5883L
#define I2C3_SCL_GPIO       GPIOA
#define I2C3_SCL_PIN        GPIO_Pin_14
#define I2C3_SDA_PIN        GPIO_Pin_13

// OLED - I2C3
#define OLED_I2C            I2C3
#define OLED_ADDR           0x3C    // 7位地址

// 罗盘 - I2C3 (共享)
#define COMPASS_I2C         I2C3
#define COMPASS_ADDR        0x0D    // 7位地址

// 初始化I2C3
void i2c3_init(void) {
    RCC_HB2PeriphClockCmd(RCC_HB2Periph_GPIOA, ENABLE);
    RCC_HB1PeriphClockCmd(RCC_HB1Periph_I2C3, ENABLE);
    
    // 配置SCL和SDA引脚
    GPIO_InitTypeDef GPIO_InitStructure;
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_13 | GPIO_Pin_14;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_OD;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    
    // I2C3配置 - 400kHz
    I2C_InitTypeDef I2C_InitStructure;
    I2C_InitStructure.I2C_ClockSpeed = 400000;
    I2C_InitStructure.I2C_Mode = I2C_Mode_I2C;
    I2C_InitStructure.I2C_DutyCycle = I2C_DutyCycle_2;
    I2C_InitStructure.I2C_OwnAddress1 = 0x00;
    I2C_InitStructure.I2C_Ack = I2C_Ack_Enable;
    I2C_InitStructure.I2C_AcknowledgedAddress = I2C_AcknowledgedAddress_7bit;
    I2C_Init(I2C3, &I2C_InitStructure);
    I2C_Cmd(I2C3, ENABLE);
}
```

### 5.3 ADC配置
```c
// 电池电压检测
#define VBAT_ADC_CHANNEL    ADC_Channel_1  // PA1
#define VBAT_DIVIDER_RATIO  6.0f
#define ADC_VREF            3.3f
#define ADC_RESOLUTION      4096.0f

float get_battery_voltage(void) {
    uint16_t adc_raw = ADC_GetConversionValue(ADC1);
    return (adc_raw * ADC_VREF / ADC_RESOLUTION) * VBAT_DIVIDER_RATIO;
}
```

## 七、常见问题与解决方案

### 7.1 SWD烧录正常
**配置**: PB8/PB9专用于SWD烧录  
**说明**: USART4 GPS 使用 PC6/PC7；SWD 仍使用 PB8/PB9，互不影响  
**优势**: 可以在线调试，无需断开传感器

### 7.2 光流数据异常
**检查项**:
1. 确认使用SPI3 (不是SPI1)
2. 检查33Ω串阻是否焊接
3. 验证CS引脚 (PC5) 控制是否正确
4. 检查squal值是否 >10

### 7.3 PA13/PA14 I2C3 罗盘说明
**现象**: 罗盘无法正常工作  
**检查项**:
1. 确认罗盘地址为 0x0D
2. 检查 I2C3 上拉电阻 (通常4.7kΩ)
3. 验证 SR2525M10D 供电正常
4. 使用 I2C 扫描程序确认设备地址
5. 检查 PA14(SCL)/PA13(SDA) 是否短路或接触不良

**旧 OLED 扫描代码已作废：PA13/PA14 当前用于 I2C3 罗盘，USART4 GPS 使用 PC6/PC7。**

**历史 I2C扫描代码（仅作旧记录）**:
```c
void i2c3_scan(void) {
    printf("Scanning I2C3 bus...\r\n");
    for (uint8_t addr = 0x08; addr < 0x78; addr++) {
        I2C_GenerateSTART(I2C3, ENABLE);
        while (!I2C_CheckEvent(I2C3, I2C_EVENT_MASTER_MODE_SELECT));
        
        I2C_Send7bitAddress(I2C3, addr << 1, I2C_Direction_Transmitter);
        if (I2C_CheckEvent(I2C3, I2C_EVENT_MASTER_TRANSMITTER_MODE_SELECTED)) {
            printf("Found device at 0x%02X\r\n", addr);
            I2C_GenerateSTOP(I2C3, ENABLE);
        }
    }
    printf("Scan complete.\r\n");
}
```

### 7.4 电池电压读数错误
**检查项**:
1. 验证分压电阻: R179=10kΩ, R180=2kΩ
2. 确认分压比为1:6
3. 检查ADC参考电压为VDD33A (3.3V)
4. 用万用表测量PA1电压进行对比

### 7.5 空闲/预留无法启动
**检查项**:
1. 确认PF5/PE0 预留引脚配置正确
2. 检查3.3V供电是否稳定
3. 验证USART8波特率 (115200)
4. 尝试复位空闲/预留 (PF5低电平100ms)
