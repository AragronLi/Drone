## 一、核心配置总结

### 1.1 MCU规格
- **完整型号**: CH32H417QEU6
- **封装**: QFN128 (14mm × 14mm, 0.4mm间距)
- **引脚数**: 128引脚
- **核心**: 双核RISC-V
  - V3F: 120MHz (传感器采集)
  - V5F: 480MHz (算法控制)
- **Flash**: V3F @ 0x00000000, V5F @ 0x00010000
- **晶振**: 25MHz HSE
- **优势**: 更多GPIO和外设接口

### 1.2 关键硬件配置

| 功能 | 接口 | 引脚 | 设备 | 备注 |
|------|------|------|------|------|
| **光流传感器** | SPI3 | PB3/4/5 + PC5(CS) | PMW3901 | 33Ω串阻 |
| **GPS** | USART4 | PC6/PC7 | SR2525M10D | NMEA, 38400 默认 |
| **罗盘** | I2C3 | PA14(SCL)/PA13(SDA) | QMC5883L | SR2525M10D 集成罗盘；旧 OLED 停用 |
| **SWD烧录** | SWD | PB8(SWCLK)/PB9(SWDIO) | WCH-Link | 独立接口 |
| **电池检测** | ADC1 | PA1 | 分压电路 | 10kΩ+2kΩ (1:6) |

## 三、I2C3共享总线配置

### 3.1 总线拓扑
```
PA14 (I2C3_SCL) ──┬── QMC5883L 罗盘 (0x0D)
PA13 (I2C3_SDA) ──┘
```

### 3.2 设备地址
- **罗盘/QMC5883L**: 0x0D (7位地址) = 0x1A (8位写) / 0x1B (8位读)
- **旧 OLED**: 0x3C，当前停用，避免占用 I2C3

### 3.3 初始化代码
```c
void i2c3_init(void) {
    // 使能时钟
    RCC_HB2PeriphClockCmd(RCC_HB2Periph_GPIOA, ENABLE);
    RCC_HB1PeriphClockCmd(RCC_HB1Periph_I2C3, ENABLE);
    
    // 配置GPIO为开漏输出
    GPIO_InitTypeDef GPIO_InitStructure;
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_13 | GPIO_Pin_14;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_OD;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    
    // 配置I2C3 - 400kHz快速模式
    I2C_InitTypeDef I2C_InitStructure;
    I2C_InitStructure.I2C_ClockSpeed = 400000;
    I2C_InitStructure.I2C_Mode = I2C_Mode_I2C;
    I2C_InitStructure.I2C_DutyCycle = I2C_DutyCycle_2;
    I2C_InitStructure.I2C_OwnAddress1 = 0x00;
    I2C_InitStructure.I2C_Ack = I2C_Ack_Enable;
    I2C_InitStructure.I2C_AcknowledgedAddress = I2C_AcknowledgedAddress_7bit;
    I2C_Init(I2C3, &I2C_InitStructure);
    I2C_Cmd(I2C3, ENABLE);
}

// I2C3总线扫描
void i2c3_scan(void) {
    printf("Scanning I2C3 bus (PA13/PA14)...\r\n");
    uint8_t found = 0;
    
    for (uint8_t addr = 0x08; addr < 0x78; addr++) {
        I2C_GenerateSTART(I2C3, ENABLE);
        while (!I2C_CheckEvent(I2C3, I2C_EVENT_MASTER_MODE_SELECT));
        
        I2C_Send7bitAddress(I2C3, addr << 1, I2C_Direction_Transmitter);
        uint32_t timeout = 1000;
        while (timeout--) {
            if (I2C_CheckEvent(I2C3, I2C_EVENT_MASTER_TRANSMITTER_MODE_SELECTED)) {
                printf("  [0x%02X] Device found", addr);
                if (addr == 0x3C) printf(" - OLED OLED旧方案");
                if (addr == 0x0D) printf(" - Compass 罗盘旧方案");
                printf("\r\n");
                found++;
                I2C_GenerateSTOP(I2C3, ENABLE);
                break;
            }
        }
        I2C_GenerateSTOP(I2C3, ENABLE);
        Delay_Us(10);
    }
    printf("Scan complete. Found %d device(s).\r\n", found);
}
```

## 四、SWD烧录配置

### 4.1 接口说明
- **引脚**: PB8 (SWCLK), PB9 (SWDIO)
- **连接器**: HDR-M_2.54-4P (4针排针)
- **线序**: GND / 3V3 / SWDIO / SWCLK
- **优势**: 独立接口，可在线调试，无需断开传感器

## 五、电源系统

### 5.1 电源树
```
VBAT (4S LiPo, 14.8~16.8V)
  │
  ├─→ SY8088AAC (U12) ──→ 5V/2A ──→ 外设供电
  │                                 (TOF, GPS, 蜂鸣器)
  │
  ├─→ SY8088AAC (U14) ──→ 5V
  │    └─→ TPS54202DDCR (U15) ──→ 3.3V (VDD33) ──→ 数字电源
  │                                                 (MCU, IMU, 光流, 数传, 空闲/预留)
  │
  └─→ MPZ1608S601ATA00 (L6) ──→ VDD33A ──→ 模拟电源
                                            (ADC, DAC, 比较器)
```

### 5.2 电池电压检测
```c
// 硬件电路: VBAT → R179(10kΩ) → PA1 → R180(2kΩ) → GND
// 分压比: 2kΩ/(10kΩ+2kΩ) = 1/6
// 滤波: C105 (100nF)

#define VBAT_ADC_CHANNEL    ADC_Channel_1
#define VBAT_DIVIDER_RATIO  6.0f
#define ADC_VREF            3.3f
#define ADC_RESOLUTION      4096.0f

float get_battery_voltage(void) {
    uint16_t adc_raw = ADC_GetConversionValue(ADC1);
    float vbat = (adc_raw * ADC_VREF / ADC_RESOLUTION) * VBAT_DIVIDER_RATIO;
    return vbat;
}

// 电压告警
void check_battery_status(float vbat) {
    if (vbat > 16.8f) {
        // 过压保护
        printf("WARNING: Battery overvoltage! %.2fV\r\n", vbat);
    } else if (vbat < 14.0f) {
        // 4S低电，强制RTL
        printf("CRITICAL: Low battery! %.2fV - RTL triggered\r\n", vbat);
        trigger_rtl();
    } else if (vbat < 14.8f) {
        // 4S低电预警
        printf("WARNING: Low battery! %.2fV\r\n", vbat);
        led_blink_fast();
    }
}
```

## 八、常见问题FAQ

### PA13/PA14 最终用途：I2C3 罗盘
**A**: 检查另一个设备的供电和连接。使用万用表测量SCL/SDA是否有3.3V上拉。确认设备地址正确 (OLED=0x3C, 罗盘=0x0D)。

### Q2: SWD烧录时提示连接失败？
**A**: 检查WCH-Link连接，确认PB8/PB9引脚。不要在代码中调用`GPIO_Remap_SWJ_Disable()`。

### Q3: 光流数据全为0？
**A**: 确认使用SPI3 (PB3/4/5)，检查CS引脚 (PC5) 控制。验证33Ω串阻焊接正确。检查squal值是否>10。

### Q4: 电池电压读数偏差大？
**A**: 用万用表测量PA1电压，应为VBAT/6。检查R179 (10kΩ) 和R180 (2kΩ) 阻值。确认ADC参考电压为3.3V。

### Q5: USART5 的 PF5/PE0 如何处理？
**A**: USART5 当前为空闲/预留接口，PF5/PE0 不接外设，软件无需初始化该串口。
